# Imágenes para la página de Liderazgo

Las imágenes deben estar en la carpeta **`assets/page_liderazgo/`** (no aquí). Nombres:

**Transferencia de conocimientos:**  
`transferencia_1.png`, `transferencia_2.png`, `transferencia_3.png`, …

**Transformación de espacios (colegios, comunidades):**  
`espacios_1.png`, `espacios_2.png`, `espacios_3.png`, …

**Lo social en el centro (reinas, tecnología, etc.):**  
`social_1.png`, `social_2.png`, `social_3.png`, …

La página usa estos archivos en los carruseles. Formato recomendado: .png o .jpg.
